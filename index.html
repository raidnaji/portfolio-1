Projects

  • Haikus Highway

    Sass | React | Firebase | API

    A React-based app that offers users to build their own haiku, a three lines short poem originated in Japan. Users can type in their own words, choose words from the suggested selection retrieved from an API, or do both. As soon as a haiku is completed, they can add their poem to the Haiku Collection which is stored in the Firebase.

    Website | GitHub

  • Soccer Statistics App

    Sass | React | API

    The Soccer App built with React presents up-to-date standings in soccer leagues using the latest API data. A user selects any particular league by clicking on a logo and will be shown requested statistics in a convenient table format.

    Website | GitHub

  • Friendly Weather App

    HTML5 | CSS3 | Sass | API

    A handy app created with Vanilla JS shows current weather conditions for a user in their location. There are two API calls made to get all the necessary data. The first call gets user coordinates based on their IP address. The second call retrieves weather information using previously obtained location data.

    Website | GitHub
